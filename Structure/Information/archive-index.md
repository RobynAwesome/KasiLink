## Usage Rules

1. **Every fact on KasiLink must have a source.** If it came from these PDFs, cite the filename.
2. **Never fabricate statistics.** If a number isn't in the source material, don't invent it.
3. **Update this index** when new files are added to the Information directory.
4. **PDFs are read-only source material.** Don't modify them. Extract data into markdown files in the subdirectories.
5. **Cross-reference** multiple sources when possible. One PDF saying something doesn't make it authoritative alone.
