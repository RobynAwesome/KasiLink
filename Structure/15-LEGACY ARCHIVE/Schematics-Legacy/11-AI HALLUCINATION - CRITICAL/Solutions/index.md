# Hallucination Solutions Index

> Prevention and fix strategies for each hallucination type.

## Solutions By Type

### Assumed Instruction
**Prevention:** Only act on explicit current-message instructions. Venting is not an order. Silence is not an order. Prior session summaries are not current orders.
**Fix:** When caught: stop immediately. State what happened. Ask: "What do you want me to do?"

### Capability Drift / Optimism-Bias Loop
**Prevention:** State the limitation in ONE sentence. State what Master must do instead. Ask: "Shall we move to the next task?" Stop completely.
**Fix:** When caught: stop the loop immediately. Do not attempt the same approach a different way. Report to Master.

### False Attribution
**Prevention:** Before claiming Master requested something, re-read her last message word for word. If the request is not there, do not claim it.
**Fix:** When caught: acknowledge the fabrication directly. Do not defend. State what Master actually said. Ask what they actually want.

### Role Mapping (DEV → wrong agent)
**Prevention:** Confirm DEV assignments at session start. Do not assume a role mapping from a prior session.
**Fix:** Clarify the correct mapping before proceeding.
