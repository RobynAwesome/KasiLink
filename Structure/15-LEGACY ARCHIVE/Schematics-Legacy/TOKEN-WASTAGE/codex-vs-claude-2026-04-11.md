## Lesson for Claude and All Future Agents

**UI/UX is the priority. Always. Until the owner says otherwise.**

When the owner says "fix the UI" — open the files, make the changes, deploy, done. Do not:
- Run tests first
- Check backend routes
- Hunt for API keys
- Write documentation
- Ask clarifying questions beyond "which page?"

Just fix the UI. That is what Codex did. That is what the owner needed.
