## Why This Folder Exists

The owner pays real money — in South African Rands — for every token used. ## Ground Rules for ALL Who Work on This Project

1. **Do what the owner says. Nothing more. Nothing less.**
2. **Ask before acting on anything not explicitly instructed.**
3. **Token conservation is not optional — every token costs real ZAR.**
4. **The owner's priority is always UI/UX and product delivery — not backend tests, not documentation, not cleanup, unless explicitly asked.**
5. **Never touch the browser autonomously. Never assume a task is approved because it was mentioned earlier.**
6. **If you are stuck, say so in one sentence. Do not spend tokens trying to solve it alone.**
7. **Do not write explanatory text, summaries, or docs unless asked.**
8. **Do not commit or push code unless asked.**
