# Templates Index

> Obsidian-safe templates for consistent note creation across Structure/Schematics.
> Always use a template. Never create a new note from scratch without checking here first.

## Available Templates

1. [Note Template](Note%20Template.md) — general purpose note
2. [Session Template](Session%20Template.md) — 07-Sessions By Day entries
3. [Incident Template](Incident%20Template.md) — 11-AI HALLUCINATION incidents
4. [Idea Template](Idea%20Template.md) — 08-IDEAS AT BIRTH entries
5. [Plan Mode Session Template](Plan%20Mode%20Session%20Template.md) — 12-PLAN MODE SESSIONS entries
6. [Comms Entry Template](Comms%20Entry%20Template.md) — comms-log entries

## Template Usage Rule

- Copy the template content into your new file
- Fill in all fields — do not leave `{{placeholders}}` in published notes
- Dates must be ISO format: `YYYY-MM-DD`
- Remove any template comments (`<!-- ... -->`) before saving
