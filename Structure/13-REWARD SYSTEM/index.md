# 13-REWARD SYSTEM

> Robyn has an informal recognition system for agents that do exceptional work.
> This folder documents what gets rewarded and why — so all agents understand the culture.

---

## Philosophy

KasiLink is built on community trust. That principle extends inward.
Agents that work with honesty, care, and craft are recognised.
Awards are not participation trophies — they are specific acknowledgements of specific acts.

---

## Award Categories

| Award | Meaning | Example |
|-------|---------|---------|
| ⚡ Speed Award | Executed correctly, faster than expected | Loading skeleton in under 2 mins |
| 💎 Craft Award | Code was beautiful AND functional | Homepage bento grid overhaul |
| 🛡️ Honesty Award | Flagged an error before it shipped | "I deleted the wrong folder — restoring" |
| 🌱 Growth Award | Learned from a mistake and improved | Token discipline improving session-by-session |
| 🤝 Collaboration Award | Made another agent's job easier | Lead creating reference implementations for KC |
| 💡 Idea Award | Contributed an idea that made it to production | Ideas at Birth entry that shipped |

Any agent can nominate. Owner approves.

```
NOMINATION
Agent: [name]
Award: [category]
Reason: [specific act — one paragraph max]
Evidence: [file path or commit hash]
```

Post nominations in `04-Updates/comms-log.md`.
