# Repo Map

## Runtime

- `app/`: URL routes and API endpoints.
- `features/gigs/`: Gig listing and creation service logic.
- `features/chat/`: Message retrieval and send service logic.
- `features/home/`: Home page data-loading service logic.
- `components/`: Reusable UI building blocks.
- `lib/`: Database, models, validation, telemetry helpers.
- `tests/`: Node test suite.

## Non-Runtime

- `docs/`: Canonical technical documentation.
- `prototypes/`: Standalone prototype source files.
- `scripts/`: Seed/utility scripts.
- `Structure/`: Working archive/design notes (non-canonical).
